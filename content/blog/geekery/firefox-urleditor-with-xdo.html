  • ffue (firefox url edit)
  • ffup (firefox url paste)
  • run-xterm (run an xterm my way)
Then just run 'ffue' and it'll open up vi with the current url split so you can edit it. Save+quit and it'll unsplit it and use ffup to go to the modified url.
