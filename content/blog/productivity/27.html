It lets you filter a given number of lines (or characters) through an external program. Here's an example: Let's say we have a file that has blank lines on it and we want to remove those blank lines? Here's what the file I made looks like: 
Hello there

How are you



Plz kthx?
So, we want those blank lines gone, right? We can use the ! command with grep to do that! Type this:
!Ggrep -v '^$'

Now we're set, the result should look like this: 
Hello there
How are you
Plz kthx?
