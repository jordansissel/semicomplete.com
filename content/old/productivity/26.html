+++ # Imported from original pyblosxom .txt format at 2015-08-14 21:10:30 -0700 date = "2003-07-07T22:16:00-07:00" title = "More vim tricks..\n" # Marked a draft because frankly my old writing may not be worth surfacing again. # I made bad word choices and had some strong-and-closed opinions years ago. # We learn over time, eh? :) draft = true type = "blog" categories = [ "blog" ] tags = ["vim"] +++ I've often wanted a command in vi and vim that'll increment a number that the cursor is over.
In nvi (freebsd), the command is #(+#|-) where ## and #+ will increment, #- will decrement. In vim the command is ^A and ^X. ^A increments while ^X decrements Both versions of vi let you add|subtract a given amount, for example: 300## will add 300 to the number. Replace ## with ^A in vim for the same effect. Nvi has a maximum of (+/-)2^31, while VIM has a range of (+/-)2^32.
